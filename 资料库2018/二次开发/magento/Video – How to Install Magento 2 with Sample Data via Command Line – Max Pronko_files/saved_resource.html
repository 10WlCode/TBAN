                    Keyur Shah —

                    In newer version of Magento 2.2.1 you OOTB Magento provides the command so you don't need to write anything at code level. To enable you just need to run the command,

                    php bin/magento dev:query-log:enable

                    and to disable you just need to run,

                    php bin/magento dev:query-log:disable

                    Keyur Shah
