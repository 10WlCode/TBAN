cmd + option + [ 折叠鼠标所在代码段
cmd + option + ] 展开鼠标所在代码段
全选之后 cmd + k 然后 cmd + num（1， 2， 3， 4）可以选择折叠级别

其实可以cmd + shift + p输入折叠查看所有和折叠相关的功能以及快捷键
