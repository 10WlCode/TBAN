          Dan Abramov‏ @dan_abramov 23 Dec 2017
          Replying to @MiltonLn @victorbarbu17 @ReactJS_News

          If you can, cancel the fetch in componentWillUnmount. If you can't, do this: componentDidMount() { this.isMounted = true; fetch(stuff).then(() => { if (this.isMounted) { this.setState(...) } }); } componentWillUnmount() { this.isMounted = false; }
