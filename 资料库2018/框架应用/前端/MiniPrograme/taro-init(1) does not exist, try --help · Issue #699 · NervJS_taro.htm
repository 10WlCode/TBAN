报错直接使用
如果是yarn 安装的 yarn global remove @tarojs/cli
使用
npm install -g npm
npm install -g @tarojs/cli

如果 有nvm
nvm install-latest-npm
nvm ls-remote
nvm install 11.6.0
nvm alias default 11.6.0
npm install -g @tarojs/cli

macOS Mojave 版本 10.14
就可以愉快的 taro init demo 了

我就是这么搞定的
